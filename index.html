About

CuteRest is an opensource and multiplateform tool dedicated to communicate with a REST server. It's particulary useful to manage Json Request during developement step. Actually, They are several tool for this purpose. You can use plugins from your browser, like Postman. or use httpie as command-line from your console. But those application doesn't provide a tree view for Json or XML response. YOu only have raw data, and you must use another software, like jsonviewer to view your content as a tree.
CuteRest have both. It's a REST client which have different view to parse the response.

Feature

TreeView

Boring to read raw data ? CuteRest render JSON or XML data as a TreeView. You can either collapse or expand interested data. CuteRest also have a Search tool to get quickly your node. For exemple you can filter data by key or by value , see what you want and extract data easily. In the next release, regular expression will be supported to help you more. There is also a color syntax highlight to warn you about each kind of variable type. For exemple string are colored in blue whereas integer are colored in orange.

Create HTTP Request

CuteRest has same feature than most of other REST client tools. And support all common Verb from the HTTP specification. You can create a HTTP request from a dedicated dialog and fill all parameter and header input. Proxy settings is also embeeded. So you can use CuteRest throught a firewall at your office. If you API Request need authentification, CuteRest support Basic and Digest HTTP authentification. Next release will have other authentification system like OAuth.

Favorite

You can save your request into the favorite tree. Those requests will be ordered by serve rname. And don't afraid to lost data, you can easily import and export data to share with collaborator or on your cloud. During your test, all requests are saved into the history view. You can then repeat request already tested without rewrite a complete request.
